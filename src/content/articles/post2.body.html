Sending emails programmatically is a common requirement in modern web applications. Whether you need to send welcome emails, password reset links, or notifications, Node.js provides excellent libraries to handle email functionality. In this tutorial, we'll explore how to send emails using Node.js with the popular Nodemailer library.

What is Nodemailer?

Nodemailer is a module for Node.js applications that allows you to send emails easily. It supports various transport methods including SMTP, sendmail, and more. It's one of the most popular and reliable email libraries for Node.js, with support for HTML emails, attachments, and embedded images.

Installing Nodemailer

First, create a new Node.js project and install Nodemailer:

npm init -y
npm install nodemailer

Step 1: Code example to send email

Create a file named sendEmail.js and add the following code:

const nodemailer = require('nodemailer')

// ===== Create transporter =====
const transporter = nodemailer.createTransport({
    host: 'smtp.gmail.com',
    port: 587,
    secure: false, // true for 465, false for other ports
    auth: {
        user: 'your-email@gmail.com',
        pass: 'your-app-password' // Use App Password for Gmail
    }
})

// ===== Email options =====
const mailOptions = {
    from: 'your-email@gmail.com',
    to: 'recipient@example.com',
    subject: 'Hello from Node.js',
    text: 'This is a plain text email sent from Node.js using Nodemailer!',
    html: '<p>Hello from Node.js!</p>' +
          '<p>This is an HTML email sent from Node.js using Nodemailer!</p>'
}

// ===== Send email =====
transporter.sendMail(mailOptions, (error, info) => {
    if (error) {
        console.log('Error occurred:', error.message)
    } else {
        console.log('Email sent successfully!')
        console.log('Message ID:', info.messageId)
    }
})

Step 2: Setting up Gmail

If you're using Gmail, you need to enable 2-Step Verification and generate an App Password:

  1. Go to your Google Account settings
  2. Navigate to Security → 2-Step Verification → App passwords
  3. Generate a new app password for "Mail"
  4. Use this 16-character password in your code

Testing the Email

Run the script with the following command:

node sendEmail.js

Output:

Email sent successfully!
Message ID: <message-id>

Using Async/Await

Here's a more modern approach using async/await:

const nodemailer = require('nodemailer')

async function sendEmail() {
    const transporter = nodemailer.createTransport({
        host: 'smtp.gmail.com',
        port: 587,
        secure: false,
        auth: {
            user: 'your-email@gmail.com',
            pass: 'your-app-password'
        }
    })

    const mailOptions = {
        from: 'your-email@gmail.com',
        to: 'recipient@example.com',
        subject: 'Test Email',
        html: '<h2>Welcome!</h2><p>Thank you for signing up.</p>'
    }

    try {
        const info = await transporter.sendMail(mailOptions)
        console.log('Email sent:', info.messageId)
    } catch (error) {
        console.error('Error sending email:', error)
    }
}
